# 최대값 구하기 (HARD)

정수 배열 `nums`와 정수 `k`가 있습니다. 배열에서 연속된 `k`개의 정수를 왼쪽에서 오른쪽 방향으로 한 칸씩 옮기며 차례로 선택할 때, 선택된 `k`개의 정수들 중 최대값들을 반환해보세요.

## 예시 1

입력값: nums = [1,3,-1,-3,5,3,6,7], k = 3
기대값: [3,3,5,5,6,7]

## 예시 2

입력값: nums = [1], k = 1
기대값: [1]

## 제한 조건

- 1 <= nums.length <= 10^5
- 10^4 <= nums[i] <= 10^4
- 1 <= k <= nums.length
